# Vib（ヴィブ）

HAL東京 Team 222 が開発した「すれ違いSNS」の公式Webサイトです。

## プロジェクト概要
BLE・GPS・Firebaseを活用した擦れ違い型ソーシャルアプリのプロモーションサイト。感情の可視化、リアルタイム共有、匿名プロフィールなどの機能を紹介しています。
