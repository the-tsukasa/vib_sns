## 📖 プロジェクト概要

**Vib（ヴィブ）** は、HAL東京 Team 222 が開発した「すれ違いSNS」の公式Webサイトです。

BLE（Bluetooth Low Energy）・GPS・Firebaseを活用した擦れ違い型ソーシャルアプリのプロモーションサイトで、感情の可視化、リアルタイム共有、匿名プロフィールなどの機能を紹介しています。

### コンセプト

日常の「すれ違い」を特別な出会いに変える。位置情報とBLE技術を組み合わせ、リアルな空間での人との繋がりを可視化するソーシャルプラットフォームです。

---

## ✨ 主な機能

- 🎯 **BLE近接検知** - すれ違った人を自動検出
- 📍 **GPS位置同期** - リアルタイムで周辺ユーザーを可視化
- 😊 **感情の可視化** - 今の気分をアバターで表現
- 🔒 **匿名プロフィール** - プライバシーを守りながら交流
- 🔥 **Firebase連携** - リアルタイムデータ同期
